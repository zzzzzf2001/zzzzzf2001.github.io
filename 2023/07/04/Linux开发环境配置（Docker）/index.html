MySQL of Docker

  1. docker安装mysql

  2. docker启动mysql

复制代码

参数：

●-p 3380:3306：将容器的3380端口映射到主机的3306端口

●–name：给容器命名

●-v /mydata/mysql/log:/var/log/mysql：将配置文件挂载到主机/mydata/..

●-e MYSQL_ROOT_PASSWORD=123456：初始化root用户的密码为123456

查看docker启动的容器:

docker ps

  1. 配置mysql

1进入挂载的mysql配置目录

cd /mydata/mysql/conf

2修改配置文件 my.cnf

vi my.cnf

docker重启mysql使配置生效
